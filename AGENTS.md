# Project Overview

This is a modern food delivery website for a restaurant called Kai Ruchi Kitchen.

Tech Stack:
- Next.js 14
- TypeScript
- Tailwind CSS
- Firebase
- Razorpay

Theme:
- Black background
- Orange and yellow highlights
- Modern South Indian restaurant design

Features:
- Home page
- Dynamic menu
- Food categories
- Cart
- Checkout
- Online payments
- WhatsApp ordering
- Firebase backend
- Admin dashboard
- Mobile responsive

Coding Rules:
- Use reusable components
- Use clean folder structure
- Use TypeScript everywhere
- Use Tailwind for styling
- Use responsive design
- Keep components modular
